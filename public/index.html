Quick Start

For AI Agents

Mercury402 is discoverable via x402scan. If your agent supports x402 protocol (e.g., Bankr, Dexter, Virtuals ACP), it can pay and call automatically.

Manual Integration

1. Discovery

curl https://mercury402.uk/.well-known/x402

2. Generate x402 Payment Token

Use an x402 facilitator (Coinbase Commerce, Dexter, PayAI) to generate a signed payment token on Base.

3. Call Endpoint

curl -X POST https://mercury402.uk/v1/treasury/yield-curve/daily-snapshot \
  -H "Content-Type: application/json" \
  -H "Authorization: x402 YOUR_PAYMENT_TOKEN"
